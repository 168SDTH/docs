# bot使用说明

注意

若无特殊说明，则仅限授权账号使用

# 杂项

# 检查bot是否在线

输入如下命令，若有消息回复，则bot正常运行（无账号限制）

/ping

# 获取IP

使用如下命令获取路由器内网IPv4以及IPv6(若连接公网则可获得公网IP)

/ipconfig

# 执行CMD命令

使用如下命令控制宿主机的CMD命令行

/cmd <需要执行的命令>

例如：

/cmd ping 127.0.0.1

# OCR文字识别

使用如下命令获取图片中的文字（无账号限制）

/ocr [图片]

提示

此功能依赖于百度文字识别api

# 获取课程表

使用此功能前需要将教务网的课程表数据以json格式存储在相应文件内

获取n天后的课程表，不加天数默认为当天

指令如下：

/class <n>

例如：获取当天课程表

/class

获取2天后的课程表

/class 2

获取3天前的课程表

/class -3

# AI聊天

# ChatGPT（非官方接口）（无账号限制）

使用下面的命令与ChatGPT对话

/aichat <你的内容>

例如：

/aichat 你好，你叫什么名字

注意

此功能不支持联系上下文

# BingAI

使用下面的命令与BingAI对话，若是首句则自动开启一轮对话

/bing <你的内容>

结束本轮对话：

/bing 结束对话

例如：

user:
/bing 你好，你可以做什么?
bot:
对话开始
bot:
(回复消息)

user:
/bing 好的，谢谢你
bot:
(回复消息)

user:
/bing 结束对话
bot:
对话已结束

注意

新版必应一轮最多支持五次对话，24小时内最多50条对话

# AI绘画（无账号限制）

# 文生图

使用如下命令获取一张AI生成的图片

/draw <steps> <prompt> negative prompt:<negative prompt>
参数 含义 数据类型
steps AI绘画迭代步数 int(1-150)
prompt 正面tags str
negative prompt 负面tags str

提示

所有参数均为非必填，不填steps数据则默认20步，不填prompt自动留空

注意事项

一定要在负面tags前加上“negative prompt”，否则会识别为正面tags

使用例如下

/draw 20 tree,cute dogs negative prompt:cats

# 获取采样器列表

使用如下命令获取获取采样器列表

/samplerlist

# 修改采样器参数

使用如下命令修改所使用的采样器

/samplerset <采样器名称>

例如

/samplerset Euler a
